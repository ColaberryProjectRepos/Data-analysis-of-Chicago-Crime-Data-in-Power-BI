Load the Data from the Website

1. Download the attached PBI file to use as a starting point. Save the file to your computer and double click to start Power BI with the report template.


2. Now we need to load data to our report. Since the data is continuously updated, its best to load the data directly via the website instead of downloading the data file locally and loading the file from the local saved file.

PLEASE NOTE THAT THIS CAN TAKE SEVERAL MINUTES

Here are the steps required to load the data from the website:

a.      Click Get Data in Power BI 

b.      Choose Web as data by clicking on "Other" and then clicking "Connect"

c.       Paste website link for the data file: https://data.cityofchicago.org/api/views/ijzp-q8t2/rows.csv?accessType=DOWNLOAD

                           d.     Load the data by clicking "OK"
