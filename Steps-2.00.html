Transform the Data

Now that we have loaded the data into Power BI, we need to transform it to better suit our needs.

1. Navigate to the "Data" tab:

 

2. Access Power Query Editor to make necessary transformations by clicking "Edit Queries" in the home menu.


3. In Power Query Editor we need to change the data type for Zip codes from number to a text so that we can showcase data on it properly. Mark the Zip Codes column and change it to Data Type: Text


4. We also need to remove the columns that we don't need to improve performance. Mark the column and click Remove Columns. Please note that this can take a long time as the data file is large.

5. Now we are ready to Close & Apply to save the changes.



6. Back in Data tab, we need to create calculated columns and measures using DAX to be able to calculate with and without filters, percentages, etc for our reports. 

You Create a Measure by clicking "Modeling" and then "New Measure"


Then you need to copy paste the DAX for the measures below. Create a measure for each DAX query a) to g)

    a) All ID = COUNTROWS(ALL('rows'[ID]))

    b)  All Zip = CALCULATE(DISTINCTCOUNT('rows'[Zip Codes]),ALL('rows'[Zip Codes]))

    c)  Arrests True Avg = DIVIDE(DIVIDE(DIVIDE('rows'[All ID], 'rows'[All Zip],0),19,0),2,0)

    d) Arrests True = CALCULATE(COUNT('rows'[Arrest]),FILTER('rows', 'rows'[Arrest]="True"))

    e) Crime Avg = DIVIDE(DIVIDE('rows'[All ID], 'rows'[All Zip],0),19,0)

    f) Total Crimes = CALCULATE(COUNTROWS(ALL('rows'[ID])),ALL('rows'[Zip Codes]))

    g) Crime Change % 2001-2019 = DIVIDE(CALCULATE(SUM('rows'[ID]),FILTER('rows',           'rows'[Year]=MAX('rows'[Year]))),CALCULATE(SUM('rows'[ID]),FILTER('rows','rows'[Year]=MIN('rows'[Year]))),0)-1
